# usherin making — 30-day content calendar

Cadence: **5 posts / week on Instagram**, **1 journal post / week on the site**, **1 story set / day** (Mon–Sat).

Pillar mix follows 40 / 25 / 25 / 10:

| Pillar | % | What it is |
|---|---|---|
| **Work** (portfolio) | 40 | Single frames, carousels, and behind-the-scenes from real shoots |
| **Place** (Okinawa) | 25 | The island itself — light, seasons, quiet corners |
| **Studio** (process) | 25 | Styling boards, moodboards, prep, booking guides |
| **People** (client + team) | 10 | Couple spotlights, team notes, replies to DMs |

All captions follow the voice rules in [brand-voice.md](./brand-voice.md): gentle, specific, spare, rooted, patient. No hype words.

## Journal post templates

All posts target ~1,200 words, use one H1 + three H2s, include the primary keyword in the title, the first paragraph, and two H2s. Follows [seo-audit.md](./seo-audit.md) keyword map.

### Template 1 — Shoot story
```
# Aya & Sho — a quiet east-coast afternoon

Okinawa wedding photography works best when the day slows down…

## Where we met
## How we styled
## What we kept (gallery)

---
If you're planning your own Okinawa wedding or couple snap, say hello.
```

### Template 2 — Seasonal guide
```
# Best time of year for Okinawa wedding photos

There are four windows we like for Okinawa wedding photography…

## Late March — bougainvillea
## Early May — blue water, long light
## Early November — susuki (pampas)
## Any calm January — quiet, empty, ours

## Booking windows
```

## Reusing this calendar month-over-month

Each month uses the same four themes but rotates the work shown.
- **Week 1** → Sunset / silhouette work
- **Week 2** → Field / landscape context
- **Week 3** → Blue-hour / night work
- **Week 4** → Warm / close work + a booking push
